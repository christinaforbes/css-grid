What is Grid?

Grid is a method to arrange web page elements in two dimensions.

Advantages over Alternatives

Before Grid was added to CSS, web developers had to lay out pages using other techniques, including tables and floats. However, these are not the best approaches. Using HTML tables is not semantically correct since it uses HTML elements to style pages instead of using CSS. Conversely, using floats is also problematic as floats were intended for positioning images on a page and allowing text to flow around them. As a result, using floats for page layout is challenging due to their number of quirks. Grid provides an easier way to style pages while keeping the HTML and CSS separate.

Grid versus Flexbox

In addition to Grid, there is also Flexbox for layout. Flexbox is a one-dimensional layout system, whereas Grid is a two-dimensional layout system. Moreover, Grid is based on the layout that the content is then placed into, while Flexbox is based on the content that then determines the layout. As a result, Grid is more suitable for larger sections of pages and Flexbox is more suitable for smaller sections of pages.

Browser Support

Currently, Grid is supported by most major browsers. However, Internet Explorer only supports an older version of Grid using a prefix, and Opera Mini does not support it.

Grid Terminology

Grid Container
A grid container is an element the declaration display: grid; is added to.
Grid Item
A grid item is a direct descendant element of a grid container.
Grid Line
A grid line is one of a number of horizontal or vertical lines that intersect to produce the grid.
Grid Track
A grid track is the area between two parallel grid lines. A grid track constitutes a row or column of the grid.
Grid Cell
A grid cell is the area where a horizontal grid track and a vertical grid track overlap.
Grid Area
A grid area is a rectangular area containing at least one grid cell.
Grid Gap
A grid gap is the empty space between two parallel grid tracks.
Explicit Grid
An explicit grid is the portion of a grid manually defined in the CSS using the grid-template-rows and grid-template-columns properties.
Implicit Grid
An implicit grid is the portion of a grid automatically generated to fit all the grid items in the grid.

Grid Properties

Creating a Grid

To create a grid, first create a grid container by adding the declaration display: grid; to the parent element of the elements to be placed in the grid.

Then, create the rows and columns of the grid using the grid-template-rows and grid-template-columns properties. These properties are used to set the number of rows and columns and the sizes of those rows and columns. The size can use any length unit (such as pixels or ems), a percentage, or the fr unit. The fr unit is a unit exclusive to Grid equal to a fraction of the remaining space in the grid.

Note, though, that using the grid-template-rows and grid-template-columns properties to create the rows and columns is not required, since the grid will generate rows and columns as needed to fit the items placed in it.

If a gap between the rows or columns is desired, use the row-gap and column-gap properties to specify the width of the gap. Alternatively, use the shorthand property, gap, to define the row gap and column gap in the same declaration.

Placing Items in the Grid

Once the grid is set up, place elements in the grid by using the grid-row-start, grid-row-end, grid-column-start, and grid-column-end properties to specify the starting grid line number and ending grid line number of the row and column, respectively. The shorthand properties, grid-row and grid-column, can also be used to define the starting grid line number and ending grid line number in the same declaration, separated by a forward slash.

Aligning Items in the Grid

Furthermore, it is possible to align items in the grid with the justify-items, justify-content, justify-self, align-items, align-content, and align-self properties.

The justify-items and align-items properties align grid items in their grid cells, the justify-content and align-content properties align the grid in its grid container if there is empty space, and the justify-self and align-self properties align an individual grid item in its grid cell.

The difference between the justify-* and the align-* properties is that the justify-* properties position items along the horizontal axis, whereas the align-* properties position items along the vertical axis.

Grid Resources

Although this was a brief overview of Grid, there are numerous online resources with additional information. Three websites for learning Grid are Grid Garden, CSS Grid PlayGround, and Grid by Example.

Grid Garden

The first resource is Grid Garden. Grid Garden is a game by Codepip that teaches Grid. The player is introduced to a concept and then must write CSS code to tend to a carrot garden. The game has twenty-eight levels and covers various properties, including grid-column-start, grid-column-end, grid-column, grid-row-start, grid-row, grid-area, order, grid-template-columns, grid-template-rows, and grid-template.

CSS Grid PlayGround

Another informative website is CSS Grid PlayGround. CSS Grid PlayGround is part of the Firefox DevTools PlayGround by Mozilla. It is a tutorial on Grid and is divided into sections. There is an emphasis on using the Firefox DevTools, which have specific features for working with a grid layout, such as displaying the grid lines, the grid line numbers, and the grid area names. The tutorial consists of Grid terminology, creating a grid, the Firefox DevTools, the fr unit, mixing units, placing items in a grid, designing a basic layout using Grid, creating named template areas, and creating named lines.

Grid by Example

Finally, there is Grid by Example. Grid by Example is a resource on Grid created by Rachel Andrew. It contains examples, videos, and links to other resources. The examples begin with a brief explanation and are followed by code demonstrating a particular concept. The videos are short tutorials on various aspects of Grid.
